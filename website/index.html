把 SQL 资产，直接变成可运行 BI 原型

SQL2BI 用流水线方式把 sql.md 转成可交互看板，覆盖 SQL 解析、语义推断、图表推荐、Dashboard 规范生成与服务联调。

核心能力

SQL 语义建模

解析多段 SQL，抽取指标、维度、过滤器，沉淀语义目录。

图表与布局自动推荐

根据字段类型与业务语义给出图表类型和 dashboard 组件排布。

前后端一体联调

自动产出可运行服务，支持 FastAPI 全量模式和 Lite 回退模式。

审计可追溯

每次查询和 SQL 执行结果可审计，产物落盘到 audit/。

架构流程

  1. 01 读取 sql.md 多 SQL 块
  2. 02 解析查询并推断语义
  3. 03 生成图表计划与看板规范
  4. 04 启动后端 API 与前端可视化
  5. 05 输出 out/ 与 audit/ 工件

快速开始

以下命令可在本地启动完整体验。

bash services/start_backend.sh
bash services/start_frontend.sh
bash services/start_skill_agent.sh
打开前端 检查后端健康
